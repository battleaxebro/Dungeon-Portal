Dungeon Portal is an HTML based tool for D&D. In remote TTRPGs all interactions outside combat are usually flat with characters looking at a jpeg on the screen or a miniature. 
Dungeon Portal proposes an alternative, with this tool DMs can setup different rooms and sprites and have their players join. They can reveal information and have it display on the screen next to their character. 

Roadmap Ideas:
Accounts and saved rooms
2-3 sprites on screen at same time and ability to switch between them 
Character creator
Sprite shop
API for roll20 and VTT foundry integration
Local App
Better Lip Syncing
